# Space Shooter

![Space Shooter](/README/README.gif)

## General info

Space Shooter is simple Ruby game that uses Gosu library. Your mission is to avoid and destroy meteorites, which follow you. With every destroyed meteorite you gain point, what makes meteorites fly faster. 

Leaderboard is a web application that is used to store, display and return high scores to game. You can access by this link: [Space Shooter Leaderboard](https://space-shooter-leaderboard.herokuapp.com/). Try to get the highest score ðŸ˜ƒ.

## How to start

```
gem bundle install
ruby ./game.rb
```
